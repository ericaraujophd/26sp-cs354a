---

This course is organized into weekly modules, each covering specific topics in SQL and database management. Each week includes lectures, readings, assignments, and quizzes to help you understand and apply the material.

- **Textbook:** *Getting Started with SQL and Databases* by Mark Simon, which is available for free using the [Hekman Library](https://calvin.primo.exlibrisgroup.com/permalink/01CALVIN_INST/nvgmcc/alma991014067486808316).
- **Lectures:** Weekly lectures will cover the main topics and provide examples. Attendance is important (and required) to grasp the material and ask questions.
